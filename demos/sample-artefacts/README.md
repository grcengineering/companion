# Sample Artefact Pack

Synthetic files for practising with The GRC Companion without bringing real work.

Use these with `/companion:retro`, a ChatGPT Project, Gemini Gem, Claude Project, Codex session, Cursor workspace, Notion Custom Agent, or Microsoft 365 Copilot declarative agent.

The pack includes:

- `sample-vendor-questionnaire.md`
- `sample-audit-walkthrough.md`
- `sample-policy-draft.md`
- `sample-control-review.md`

Each file is fictional. The rough edges are intentional so the Companion has something to teach from.
