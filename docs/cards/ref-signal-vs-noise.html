Signal vs. Noise

Reading reference T5: Outcomes Over Activity Approved

Read this when the learner is confusing activity with impact. The piece sharpens the distinction between evidence volume and useful assurance signal, which supports the primitive that GRC should reduce translation loss and improve decisions rather than simply produce more compliance motion. It is a strong companion reference for coaching learners to ask what a metric changes, not just what it counts.
