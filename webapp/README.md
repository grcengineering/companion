# Webapp

Tier 0 webapp source notes.

The deployable GitHub Pages output lives in `docs/` because this repository currently publishes from the `/docs` folder.

Sprint 1 is static and local-first:

- no API key in browser code
- no server-side authentication
- learner profile stored in localStorage
- progress stored in localStorage
- generated companion prompts can be copied into the learner's AI tool

Future signed-in mode needs a backend or managed auth/storage provider.
